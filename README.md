# Mental-Warrior-Demo

Mental Warrior is a Flutter self-improvement app designed to help users build discipline, improve focus, and track personal growth.

This repository now includes a starter **demo view scaffold** so you can quickly fill in your own UI and content.

## Included demo scaffold

- `pubspec.yaml` - Flutter project dependencies and metadata
- `lib/main.dart` - app entrypoint + theme setup
- `lib/screens/demo_home_view.dart` - placeholder demo home screen with core app sections
- `test/widget_test.dart` - basic smoke test for the demo UI

## Next step

Install Flutter locally, then run:

```bash
flutter pub get
flutter run
flutter test
```
